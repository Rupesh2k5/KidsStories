# 🌟 KidsStories E-Commerce Platform
A fully functional, responsive, and beautifully designed MERN stack e-commerce application tailored specifically for selling children's digital storybooks, bundles, and printable activity books. 
## ✨ Key Features
* **🛒 Seamless Shopping Experience:** A responsive, interactive UI built with React featuring glassmorphism design, smooth animations, and a dedicated shopping cart system.
* **💳 Automated Digital Delivery:** Integrated with Razorpay for secure payments. Upon successful checkout, the Node.js backend automatically emails the customer their purchased digital PDF books via Nodemailer.
* **🔐 Role-Based Admin Dashboard:** Secure owner-only dashboard to completely manage the store. Admins can track total revenue, view recent bookings, monitor weekend sales, and dynamically upload new books (with cover images) directly to the website.
* **📱 Fully Responsive:** Carefully optimized CSS ensures the website and admin dashboard look perfect on desktops, tablets, and mobile phones.
* **☁️ Cloud Image Storage:** Integrated with ImageKit for fast, optimized, and secure delivery of book cover images.
## 🛠️ Tech Stack
* **Frontend:** React.js (Vite), Context API, CSS3, Lucide React Icons
* **Backend:** Node.js, Express.js
* **Database:** MongoDB (Mongoose)
* **Authentication:** JWT (JSON Web Tokens) & bcrypt
* **Integrations:** Razorpay (Payments), Nodemailer (Automated Emails), ImageKit (Image Hosting)
